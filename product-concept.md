# 🧊 Fridge Buddy - プロダクト企画書

*最終更新: 2025-01-18*

## 📋 プロダクト概要

| 項目 | 内容 |
|------|------|
| **プロダクト名** | Fridge Buddy |
| **ミッション** | 冷蔵庫に「常に置いておきたい食材」の在庫を、写真×AIで自動管理し、買い忘れをゼロにする |
| **想定ユーザー** | 共働き世帯・子育て家庭・一人暮らしの忙しい人 |
| **開発ステータス** | ✅ **本番リリース済み**（MVP完了・実用レベル） |

---

## 🔄 核心フロー（実装済み）

| ステップ | ユーザー操作 | アプリ処理 | 出力 |
|---------|-------------|-----------|------|
| **0** | "常備食材リスト"を一度だけ登録 | Supabaseに永続保存 | 管理対象リスト保存 |
| **1** | 冷蔵庫内部を撮影（1-3角度） | 画像取得・base64変換 | 写真データ |
| **2** | ― | GPT-4oが画像解析し、登録食材を認識 | 検出食材・残量・信頼度 |
| **3** | 検出結果を確認・編集 | 不足分・少量分を自動計算 | 「買うべき物リスト」 |
| **4** | リストを見て買い物<br>・タップでチェックオフ<br>・進捗確認 | 購入状況をリアルタイム同期 | 購入完了／統計更新 |

## ⭐ 実装済み機能

### 🔐 認証システム
- Supabase Authによる安全なユーザー管理
- メール認証・セッション管理・ログアウト機能

- ユーザーが食材名を登録・編集・削除

### 🤖 AI画像解析
- **GPT-4o Vision API**による高精度食材認識

- 食材名・残量レベル・信頼度の検出
- リアルタイム分析状況表示

### 🛒 モバイル最適化買い物リスト
- **外出先での使いやすさ**を重視したUI
- タップによるチェックオフ機能
- 進捗統計（残り・購入済み・完了率）
- 手動追加・削除・一括クリア機能

### 📱 レスポンシブデザイン
- スマートフォン・タブレット完全対応
- タッチデバイス最適化

## ⚠️ 技術的課題と対策

### 📸 画像認識精度
- **現状**: GPT-4oにより実用レベルの精度を実現
- **課題**: 照明・重なり・パッケージ違いへの対応
- **対策**: ユーザーフィードバック収集・Fine-Tuning活用

### 👤 ユーザー体験
- **現状**: 直感的UI・ガイダンス表示で初期負荷を軽減
- **課題**: 常備食材登録の簡素化
- **対策**: よく使われる食材の提案・音声入力検討

### 🔗 API制限
- **現状**: OpenAI API制限内で安定動作
- **課題**: 利用者増加時のスケーラビリティ
- **対策**: レート制限管理・キャッシュ活用・複数プロバイダー検討

### 🔒 プライバシー
- **現状**: 画像はAPI送信のみ・ローカル保存なし
- **課題**: より厳格なプライバシー要求への対応
- **対策**: エッジAI・ローカル処理オプションの検討
