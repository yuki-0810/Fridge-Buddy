# Fridge Buddy

あなたの冷蔵庫を賢く管理するVue.jsアプリケーション

## 機能

- 食材管理
- 賞味期限チェック
- レシピ提案
- レスポンシブデザイン

## 技術スタック

- Vue.js 3
- Vite
- CSS3（レスポンシブ対応）

## 開発環境のセットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build
```
